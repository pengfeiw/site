从 B 样条曲线到 B 样条曲面：张量积构造、计算方法与性质 #

B 样条曲面并不是一套与 B 样条曲线完全不同的理论。

它可以理解为：在两个参数方向上分别使用 B 样条基函数，再将两个方向的基函数相乘，形成一个二维张量积结构。

从 B 样条曲线过渡到 B 样条曲面，核心变化可以概括为：

B 样条曲线B 样条曲面
一个参数 uuu两个参数 u,vu,vu,v
一列控制点 PiP_iPi​二维控制点网格 Pi,jP_{i,j}Pi,j​
一个节点向量 UUU两个节点向量 U,VU,VU,V
一个次数 ppp两个次数 p,qp,qp,q
基函数 Ni,p(u)N_{i,p}(u)Ni,p​(u)乘积基函数 Ni,p(u)Nj,q(v)N_{i,p}(u)N_{j,q}(v)Ni,p​(u)Nj,q​(v)
控制多边形控制网格

一、从 B 样条曲线过渡到 B 样条曲面 #

1. B 样条曲线回顾 #

一条 ppp 次 B 样条曲线定义为

C(u)=∑i=0nNi,p(u)Pi.C(u)=\sum_{i=0}^{n}N_{i,p}(u)P_i.C(u)=i=0∑n​Ni,p​(u)Pi​.

其中：

  • uuu 是曲线参数；
  • PiP_iPi​ 是控制点；
  • Ni,p(u)N_{i,p}(u)Ni,p​(u) 是 ppp 次 B 样条基函数；
  • 控制点共有 n+1n+1n+1 个。

节点向量为

U=u0,u1,…,ur,U={u_0,u_1,\ldots,u_r},U=u0​,u1​,…,ur​,

节点数量满足

r=n+p+1.r=n+p+1.r=n+p+1.

对于一个固定的参数 uuu，曲线点 C(u)C(u)C(u) 是控制点 PiP_iPi​ 的加权组合。

2. 将控制点扩展为控制点网格 #

曲线的控制点只有一个下标：

P0,P1,…,Pn.P_0,P_1,\ldots,P_n.P0​,P1​,…,Pn​.

曲面有两个参数方向，因此控制点需要排列成二维网格：

Pi,j,i=0,…,n,j=0,…,m.P_{i,j}, \qquad i=0,\ldots,n, \quad j=0,\ldots,m.Pi,j​,i=0,…,n,j=0,…,m.

控制点网格可以写成

P0,0P0,1⋯P0,m P1,0P1,1⋯P1,m ⋮⋮⋮ Pn,0Pn,1⋯Pn,m.\begin{matrix} P_{0,0}&P_{0,1}&\cdots&P_{0,m}\ P_{1,0}&P_{1,1}&\cdots&P_{1,m}\ \vdots&\vdots&&\vdots\ P_{n,0}&P_{n,1}&\cdots&P_{n,m} \end{matrix}.P0,0​​P0,1​​⋯​P0,m​ P1,0​​P1,1​​⋯​P1,m​ ⋮​⋮​​⋮ Pn,0​​Pn,1​​⋯​Pn,m​​.

因此，从曲线到曲面的第一个变化是

控制多边形⟶控制网格\boxed{ \text{控制多边形} \longrightarrow \text{控制网格} }控制多边形⟶控制网格​

3. 把曲线的控制点替换成另一方向的曲线 #

原来的曲线为

C(u)=∑i=0nNi,p(u)Pi.C(u)=\sum_{i=0}^{n}N_{i,p}(u)P_i.C(u)=i=0∑n​Ni,p​(u)Pi​.

现在不再让 PiP_iPi​ 是固定点，而是让它随着另一个参数 vvv 变化。

对于固定的 iii，定义一条 vvv 方向的 qqq 次 B 样条曲线：

Qi(v)=∑j=0mNj,q(v)Pi,j.Q_i(v)=\sum_{j=0}^{m}N_{j,q}(v)P_{i,j}.Qi​(v)=j=0∑m​Nj,q​(v)Pi,j​.

然后将曲线公式中的 PiP_iPi​ 替换为 Qi(v)Q_i(v)Qi​(v)：

S(u,v)=∑i=0nNi,p(u)Qi(v).S(u,v)=\sum_{i=0}^{n}N_{i,p}(u)Q_i(v).S(u,v)=i=0∑n​Ni,p​(u)Qi​(v).

代入 Qi(v)Q_i(v)Qi​(v)：

S(u,v)======∑i=0nNi,p(u)(∑j=0mNj,q(v)Pi,j).S(u,v) ====== \sum_{i=0}^{n} N_{i,p}(u) \left( \sum_{j=0}^{m}N_{j,q}(v)P_{i,j} \right).S(u,v)======i=0∑n​Ni,p​(u)(j=0∑m​Nj,q​(v)Pi,j​).

展开可得

S(u,v)=∑i=0n∑j=0mNi,p(u)Nj,q(v)Pi,j\boxed{ S(u,v)= \sum_{i=0}^{n} \sum_{j=0}^{m} N_{i,p}(u)N_{j,q}(v)P_{i,j} }S(u,v)=i=0∑n​j=0∑m​Ni,p​(u)Nj,q​(v)Pi,j​​

这就是张量积 B 样条曲面的定义。

4. 为什么两个方向的基函数要相乘 #

控制点 Pi,jP_{i,j}Pi,j​ 对曲面点 S(u,v)S(u,v)S(u,v) 的影响，需要同时考虑两个方向：

  • uuu 方向的权重为 Ni,p(u)N_{i,p}(u)Ni,p​(u)；
  • vvv 方向的权重为 Nj,q(v)N_{j,q}(v)Nj,q​(v)。

因此，总权重是两个权重的乘积：

Bi,j(u,v)============Ni,p(u)Nj,q(v)\boxed{ B_{i,j}(u,v) ============ N_{i,p}(u)N_{j,q}(v) }Bi,j​(u,v)============Ni,p​(u)Nj,q​(v)​

这个二维基函数称为张量积基函数。

可以直观地理解为：

一个控制点只有在 uuu 方向和 vvv 方向都对当前参数点有影响时，才会对曲面点产生影响。

二、曲面的节点向量与参数域 #

1. 两个节点向量 #

曲面有两个参数方向，因此也有两个节点向量。

uuu 方向的节点向量为

U=u0,u1,…,ur,U={u_0,u_1,\ldots,u_r},U=u0​,u1​,…,ur​,

vvv 方向的节点向量为

V=v0,v1,…,vs.V={v_0,v_1,\ldots,v_s}.V=v0​,v1​,…,vs​.

节点数关系分别为

r=n+p+1\boxed{r=n+p+1}r=n+p+1​

和

s=m+q+1.\boxed{s=m+q+1}.s=m+q+1​.

其中：

  • n+1n+1n+1 是 uuu 方向控制点数；
  • m+1m+1m+1 是 vvv 方向控制点数；
  • ppp 是 uuu 方向次数；
  • qqq 是 vvv 方向次数。

2. 夹持节点向量 #

若两个方向都采用归一化夹持节点向量，则可以写成

U=0,…,0⏟∗p+1,u∗p+1,…,ur−p−1,1,…,1⏟p+1,U= { \underbrace{0,\ldots,0}*{p+1}, u*{p+1},\ldots,u_{r-p-1}, \underbrace{1,\ldots,1}_{p+1} },U=0,…,0​∗p+1,u∗p+1,…,ur−p−1​,p+11,…,1​​, V=0,…,0⏟∗q+1,v∗q+1,…,vs−q−1,1,…,1⏟q+1.V= { \underbrace{0,\ldots,0}*{q+1}, v*{q+1},\ldots,v_{s-q-1}, \underbrace{1,\ldots,1}_{q+1} }.V=0,…,0​∗q+1,v∗q+1,…,vs−q−1​,q+11,…,1​​.

3. 参数域 #

B 样条曲线的参数域是一个区间，而 B 样条曲面的参数域是两个区间的笛卡尔积：

(u,v)∈[up,ur−p]×[vq,vs−q]\boxed{ (u,v)\in [u_p,u_{r-p}] \times [v_q,v_{s-q}] }(u,v)∈[up​,ur−p​]×[vq​,vs−q​]​

等价于

up≤u≤ur−p,u_p\leq u\leq u_{r-p},up​≤u≤ur−p​, vq≤v≤vs−q.v_q\leq v\leq v_{s-q}.vq​≤v≤vs−q​.

如果两个方向都归一化为 [0,1][0,1][0,1]，则参数域为

(u,v)∈[0,1]×[0,1]\boxed{ (u,v)\in[0,1]\times[0,1] }(u,v)∈[0,1]×[0,1]​

这个矩形存在于 uvuvuv 参数平面中。

需要注意：

参数域是矩形，并不意味着映射到三维空间中的曲面也是矩形。

三、固定一个参数：等参数曲线 #

B 样条曲面的一个重要性质是：固定一个参数后，得到的仍然是 B 样条曲线。

1. 固定 u=u0u=u_0u=u0​ #

从曲面公式出发：

S(u,v)=∑i=0n∑j=0mNi,p(u)Nj,q(v)Pi,j.S(u,v)= \sum_{i=0}^{n}\sum_{j=0}^{m} N_{i,p}(u)N_{j,q}(v)P_{i,j}.S(u,v)=i=0∑n​j=0∑m​Ni,p​(u)Nj,q​(v)Pi,j​.

令 u=u0u=u_0u=u0​，则

S(u0,v)=∑j=0mNj,q(v)(∑i=0nNi,p(u0)Pi,j).S(u_0,v) = \sum_{j=0}^{m} N_{j,q}(v) \left( \sum_{i=0}^{n} N_{i,p}(u_0)P_{i,j} \right).S(u0​,v)=j=0∑m​Nj,q​(v)(i=0∑n​Ni,p​(u0​)Pi,j​).

Qj(u0)=∑i=0nNi,p(u0)Pi,j,Q_j(u_0) = \sum_{i=0}^{n} N_{i,p}(u_0)P_{i,j},Qj​(u0​)=i=0∑n​Ni,p​(u0​)Pi,j​,

它是一条定义在节点向量 VVV 上的 qqq 次 B 样条曲线。

2. 固定 v=v0v=v_0v=v0​ #

类似地，

因此：

固定 B 样条曲面的任意一个参数，都会得到一条 B 样条曲线。\boxed{ \text{固定 B 样条曲面的任意一个参数，都会得到一条 B 样条曲线。} }固定 B 样条曲面的任意一个参数，都会得到一条 B 样条曲线。​

这些曲线称为等参数曲线。

曲面可以看成由两族纵横交错的等参数曲线编织而成。

四、曲面点的计算方法 #

1. 计算一个曲面点的五个步骤 #

对于一个给定的参数点 (u,v)(u,v)(u,v)，计算 S(u,v)S(u,v)S(u,v) 可以分成五步。

第一步：确定 uuu 所在节点区间 #

寻找 iii，使得

u∈[ui,ui+1).u\in[u_i,u_{i+1}).u∈[ui​,ui+1​).

第二步：计算 uuu 方向非零基函数 #

只需计算

Ni−p,p(u),…,Ni,p(u).N_{i-p,p}(u),\ldots,N_{i,p}(u).Ni−p,p​(u),…,Ni,p​(u).

总共至多有 p+1p+1p+1 个。

第三步：确定 vvv 所在节点区间 #

寻找 jjj，使得

v∈[vj,vj+1).v\in[v_j,v_{j+1}).v∈[vj​,vj+1​).

第四步：计算 vvv 方向非零基函数 #

只需计算

Nj−q,q(v),…,Nj,q(v).N_{j-q,q}(v),\ldots,N_{j,q}(v).Nj−q,q​(v),…,Nj,q​(v).

总共至多有 q+1q+1q+1 个。

第五步：与局部控制点相乘并求和 #

曲面点只需使用局部控制点：

S(u,v)=∑k=i−pi∑l=j−qjNk,p(u)Nl,q(v)Pk,l\boxed{ S(u,v)= \sum_{k=i-p}^{i} \sum_{l=j-q}^{j} N_{k,p}(u)N_{l,q}(v)P_{k,l} }S(u,v)=k=i−p∑i​l=j−q∑j​Nk,p​(u)Nl,q​(v)Pk,l​​

因此，计算一个曲面点不需要遍历全部控制点，只需要至多

(p+1)(q+1)\boxed{(p+1)(q+1)}(p+1)(q+1)​

个局部控制点。

2. 矩阵形式 #

将 uuu 方向的非零基函数写成行向量：

NuT=[Ni−p,p(u)⋯Ni,p(u)].\mathbf N_u^T= \begin{bmatrix} N_{i-p,p}(u)&\cdots&N_{i,p}(u) \end{bmatrix}.NuT​=[Ni−p,p​(u)​⋯​Ni,p​(u)​].

将 vvv 方向的非零基函数写成列向量：

Nv=[Nj−q,q(v) ⋮ Nj,q(v)].\mathbf N_v= \begin{bmatrix} N_{j-q,q}(v)\ \vdots\ N_{j,q}(v) \end{bmatrix}.Nv​=[Nj−q,q​(v) ⋮ Nj,q​(v)​].

对应的局部控制点组成矩阵：

P=[Pi−p,j−q⋯Pi−p,j ⋮⋮ Pi,j−q⋯Pi,j].\mathbf P= \begin{bmatrix} P_{i-p,j-q}&\cdots&P_{i-p,j}\ \vdots&&\vdots\ P_{i,j-q}&\cdots&P_{i,j} \end{bmatrix}.P=[Pi−p,j−q​​⋯​Pi−p,j​ ⋮​​⋮ Pi,j−q​​⋯​Pi,j​​].

于是

S(u,v)=NuTPNv\boxed{ S(u,v)= \mathbf N_u^T\mathbf P\mathbf N_v }S(u,v)=NuT​PNv​​

其维度关系为

1×(p+1),1\times(p+1),1×(p+1), (p+1)×(q+1),(p+1)\times(q+1),(p+1)×(q+1), (q+1)×1.(q+1)\times1.(q+1)×1.

3. 分两次曲线求值 #

实际计算时，也可以先沿一个方向计算临时点，再沿另一个方向计算。

例如先沿 uuu 方向：

Ql(u)=∑k=i−piNk,p(u)Pk,l,Q_l(u) = \sum_{k=i-p}^{i} N_{k,p}(u)P_{k,l},Ql​(u)=k=i−p∑i​Nk,p​(u)Pk,l​,

也可以先沿 vvv 方向再沿 uuu 方向，结果完全相同。

五、张量积 B 样条基函数的性质 #

令二维基函数为

Bi,j(u,v)=Ni,p(u)Nj,q(v).B_{i,j}(u,v) = N_{i,p}(u)N_{j,q}(v).Bi,j​(u,v)=Ni,p​(u)Nj,q​(v).

这些性质大多由两个方向的一维 B 样条基函数直接推导得到。

1. 非负性 #

一维 B 样条基函数满足

Ni,p(u)≥0,N_{i,p}(u)\geq0,Ni,p​(u)≥0, Nj,q(v)≥0.N_{j,q}(v)\geq0.Nj,q​(v)≥0.

因此乘积满足

Ni,p(u)Nj,q(v)≥0\boxed{ N_{i,p}(u)N_{j,q}(v)\geq0 }Ni,p​(u)Nj,q​(v)≥0​

二维基函数不会产生负权重。

2. 规范性或单位分解 #

两个方向分别满足

∑i=0nNi,p(u)=1,\sum_{i=0}^{n}N_{i,p}(u)=1,i=0∑n​Ni,p​(u)=1, ∑j=0mNj,q(v)=1.\sum_{j=0}^{m}N_{j,q}(v)=1.j=0∑m​Nj,q​(v)=1.

因此

∑i=0n∑j=0mNi,p(u)Nj,q(v)=(∑i=0nNi,p(u))(∑j=0mNj,q(v)) =1⋅1 =1.\begin{aligned} \sum_{i=0}^{n}\sum_{j=0}^{m} N_{i,p}(u)N_{j,q}(v) &= \left( \sum_{i=0}^{n}N_{i,p}(u) \right) \left( \sum_{j=0}^{m}N_{j,q}(v) \right)\ &=1\cdot1\ &=1. \end{aligned}i=0∑n​j=0∑m​Ni,p​(u)Nj,q​(v)​=(i=0∑n​Ni,p​(u))(j=0∑m​Nj,q​(v)) ​=1⋅1 ​=1.​

所以

∑i=0n∑j=0mNi,p(u)Nj,q(v)=1\boxed{ \sum_{i=0}^{n}\sum_{j=0}^{m} N_{i,p}(u)N_{j,q}(v)=1 }i=0∑n​j=0∑m​Ni,p​(u)Nj,q​(v)=1​

3. Bézier 基函数是特殊情况 #

如果

n=p,m=q,n=p, \qquad m=q,n=p,m=q,

并且两个方向都没有内部节点：

张量积 B 样条基函数便退化为张量积 Bernstein 基函数。

4. 局部支撑性 #

一维基函数的支撑域分别为

Ni,p(u)=0,u∉[ui,ui+p+1),N_{i,p}(u)=0, \qquad u\notin[u_i,u_{i+p+1}),Ni,p​(u)=0,u∈/[ui​,ui+p+1​), Nj,q(v)=0,v∉[vj,vj+q+1).N_{j,q}(v)=0, \qquad v\notin[v_j,v_{j+q+1}).Nj,q​(v)=0,v∈/[vj​,vj+q+1​).

6. 唯一最大值 #

9. 曲面的连续性与可微性 #

曲面

的连续性与可微性由基函数决定。

  • 增加 uuu 方向节点重复度，会降低跨越对应 uuu 节点线时的光滑性；
  • 增加 vvv 方向节点重复度，会降低跨越对应 vvv 节点线时的光滑性。

10. 重复节点与重复控制点的区别 #

产生尖锐特征有两种不同方式。

重复节点 #

重复节点会直接降低基函数和曲面的参数连续阶数。

例如三次方向 p=3p=3p=3：

  • 简单节点：C2C^2C2；
  • 二重节点：C1C^1C1；
  • 三重节点：C0C^0C0。

重复控制点 #

如果相邻的若干控制点重合，可以形成明显的尖锐隆起、脊线或皱折。

但需要注意：

重复控制点不一定改变基函数本身的连续阶数。

曲面在参数意义上可能仍然具有较高阶连续性，但几何外观上可以非常尖锐。

七、曲线性质不能全部机械推广到曲面 #

多数 B 样条曲线性质都可以通过张量积推广到曲面，例如：

  • 非负性；
  • 单位分解；
  • 局部性；
  • 凸包性；
  • 仿射不变性；
  • 连续性；
  • 局部修改性。

但是，并不是所有曲线性质都能直接推广。

书中特别指出：

B 样条曲面一般不具有与 B 样条曲线完全对应的变差减小性。\boxed{ \text{B 样条曲面一般不具有与 B 样条曲线完全对应的变差减小性。} }B 样条曲面一般不具有与 B 样条曲线完全对应的变差减小性。​

对于曲线，可以利用控制多边形与直线的交点数来研究曲线的振荡程度。

但对于二维曲面，并不存在一个同样简单、直接对应的变差减小结论。

九、总结 #

B 样条曲面是 B 样条曲线在两个参数方向上的张量积推广。

从曲线

C(u)=∑i=0nNi,p(u)PiC(u)= \sum_{i=0}^{n}N_{i,p}(u)P_iC(u)=i=0∑n​Ni,p​(u)Pi​

出发，将控制点替换为另一方向上的 B 样条曲线，就得到

S(u,v)=∑i=0n∑j=0mNi,p(u)Nj,q(v)Pi,j\boxed{ S(u,v)= \sum_{i=0}^{n}\sum_{j=0}^{m} N_{i,p}(u)N_{j,q}(v)P_{i,j} }S(u,v)=i=0∑n​j=0∑m​Ni,p​(u)Nj,q​(v)Pi,j​​

理解 B 样条曲面时，可以抓住以下几个核心观点：

  1. 曲面有两个参数。

    uuu 和 vvv 分别控制两个参数方向。

  2. 控制点由一列变成二维网格。

    控制多边形推广为控制网格。

  3. 二维基函数是两个一维基函数的乘积。

    Bi,j(u,v)=Ni,p(u)Nj,q(v).B_{i,j}(u,v)=N_{i,p}(u)N_{j,q}(v).Bi,j​(u,v)=Ni,p​(u)Nj,q​(v).

  4. 固定一个参数，就得到一条 B 样条曲线。

    曲面由两族等参数曲线组成。

  5. 局部曲面点只受有限控制点影响。

    在一个节点小矩形中，最多只有

    (p+1)(q+1)(p+1)(q+1)(p+1)(q+1)

    个控制点参与计算。

  6. 曲线的多数性质可以推广到曲面。

    包括非负性、规范性、凸包性、仿射不变性、局部修改性和连续性。

  7. 连续性可以在两个方向分别控制。

    uuu 方向由次数 ppp 和节点向量 UUU 中的节点重复度决定，vvv 方向由次数 qqq 和节点向量 VVV 中的节点重复度决定。

因此，B 样条曲面可以看成：

用两个方向的 B 样条曲线，通过张量积编织而成的曲面。\boxed{ \text{用两个方向的 B 样条曲线，通过张量积编织而成的曲面。} }用两个方向的 B 样条曲线，通过张量积编织而成的曲面。​
